## 一、设计概念：从党群服务中心到邻里汇的渐变转译

### 1.1 兴安广场「渐变」手法的乡村再译

湖北宜昌兴安广场（HID 翰地景观）的核心手法是 **三道渐变**——铺装渐变、家具密度渐变、活动强度渐变——模糊了公共空间与停留空间的边界。五福村方案将其转译为线性街角绿地的「六树渐境」，本方案则面对一个不同的问题：

| 对比维度 | 五福村·六树渐境 | 三角村·邻里汇 |
|---|---|---|
| 场地属性 | 沿路带状绿地（长条形） | 党群服务中心旁口袋公园（面状） |
| 面积 | 约 160㎡ | 约 300㎡ |
| 核心功能 | 路边歇脚 → 树下社交 | 办事等候 → 邻里社交 → 文化感知 |
| 人流特征 | 线性路过，需制造停留理由 | 目的性到达 + 偶然路过，需分流等候与休闲 |
| 建筑关系 | 无相邻建筑 | 紧邻党群服务中心西门，需承接建筑外溢功能 |
| 关键约束 | 6 棵保留乔木沿路排列 | 相对方正，有集中绿化基础 |

### 1.2 设计概念说明（约 200 字）

> **一个三角形的小公园，三个方向的人走过来，谁也不急着走。**
>
> 三角村的名字里就有"三角"。我们把"三角"变成空间的语言：三条铺装带从党群服务中心西门向外辐射——一条通向村路边的树荫棋桌，一条通向草坪上的邻里廊架，一条通向花池围合的童趣角。三条线交叉的地方，是一棵保留的老榕树和一圈蚝壳水磨石坐凳。
>
> 来办事的阿叔在廊架下翻报纸等叫号，路过的阿婆在树荫棋盘边放下菜篮观战，放学的小学生在童趣角认斗门本土植物——一个小公园同时装着党群服务的效率、邻里日常的温度和斗门本土文化的厚度。
>
> 蚝壳白的铺地、青砖灰的矮墙、水蓝色的廊架顶棚、莲雾粉的标识牌……六种斗门本土颜色安静地散落在三角网格里。三角村的这个小公园，是党群服务中心的"户外客厅"，更是全村人碰面、聊天、歇脚、等办事的那棵"大树下"。

### 1.3 核心策略：三环渐变 × 三角网格

```
                    村路
                     │
    ┌────────────────┼────────────────┐
    │                │                │
    │   环一·迎候环   │  党群众服务中心 │
    │   (透水砖)     │     西门        │
    │                │                │
    │   环二·社交环   │   环三·沉浸环   │
    │   (碎石+砖)    │   (草坪+木屑)   │
    │                │                │
    └────────────────┴────────────────┘
```

| 环 | 距建筑/路边 | 铺装 | 活动类型 | 公共性 |
|---|---|---|---|---|
| **环一·迎候环** | 紧邻西门 + 沿路侧 | 浅灰白透水混凝土砖（仿蚝壳色） | 短暂等候、看公告栏、取宣传册 | ★★★★★ 全公共 |
| **环二·社交环** | 中间圈层 | 白麻碎石 + 草缝 + 局部蚝壳水磨石 | 下棋、聊天、喝茶、小型集会 | ★★★☆☆ 半公共 |
| **环三·沉浸环** | 内侧/远离建筑 | 草坪 + 树皮木屑 + 花池 | 静坐、亲子、植物认知 | ★★☆☆☆ 半私密 |

> **同款兴安广场手法**：三环之间不设任何硬质边界——没有围栏、没有路牙。铺装的纹理密度和家具的围合度是唯一的分界语言。从西门出来，你甚至注意不到自己已经从"办事"模式切换到了"休闲"模式。

## 二、斗门本土色彩基因提取

| 色彩基因 | 来源 | 色值参考 | 空间中的角色 |
|---|---|---|---|
| **蚝壳白** | 斗门传统蚝壳墙 | 暖灰白 `#F2EFE9` | 铺装主色、廊架立柱 |
| **青砖灰** | 岭南传统青砖民居 | 冷灰 `#7B8B7A` | 矮墙/花池收边/信息栏骨架 |
| **老木褐** | 渔舟/农具/旧船板 | 深褐 `#8B6914` 系 | 所有木质坐面/长凳/廊架顶棚格栅 |
| **水色蓝绿** | 河涌/稻田/蕉林 | `#5B8C7A` → `#3C6E71` | 廊架顶棚帆布/标识系统/遮阳伞 |
| **疍家土黄** | 斗门沙田/竹编/稻作 | `#C9A96E` | 碎石铺装暖色颗粒/竹编灯具 |
| **莲雾粉** | 斗门特产莲雾果花色 | `#E8A0B4` | Logo/公告栏标题/文创跳色/花池花卉 |

> **色彩渐变逻辑**：从党群服务中心西门（蚝壳白，最"正式"）→ 向外圈逐步混入更多土黄和蓝绿（越"放松"越"日常"），就像从"办事窗口"走进"村口大树下"。

## 三、场地解读与空间诊断

### 3.1 场地基本参数（基于三角村党群服务中心西侧公园现状推估）

| 参数 | 估算值 |
|---|---|
| 面积 | 约 300㎡ |
| 形状 | 近方形/矩形，位于建筑西侧 |
| 东侧 | 紧邻党群服务中心西门（建筑出入口） |
| 北侧 | 村路（宽约 4-5m），路对面为民居 |
| 西侧/南侧 | 场地内边界（围墙/相邻地块/空地——需实地核实） |
| 现有植被 | 推测保留乔木 3-5 棵（榕树/果树），部分灌木 |
| 现有设施 | 可能已有简易铺装、零星座椅或公告栏 |

### 3.2 现状问题 → 设计对策

| 现状问题 | 设计对策 |
|---|---|
| 紧邻党群服务中心，办事等候人群无处可坐 | 西门出口 5m 范围内设「候事廊」——遮阳避雨 + 长凳 + 公告栏一体化 |
| 公园与服务中心功能割裂，各用各的 | 「三环渐变」将建筑外溢功能（等候、信息获取）自然过渡为公园休闲功能 |
| 现有绿化多为"不可进入"的灌木丛 | 保留乔木，将灌木丛改造为可进入的草坪 + 碎石子坐憩区 + 抬升种植池 |
| 缺少儿童友好空间 | 增设「童趣角」——斗门本土植物认知花池 + 自然游乐元素 |
| 村路界面生硬（围墙/灌木遮挡视线） | 沿路界面打开，透水砖从路边"渗入"公园——路人看见里面的热闹，自然走进来 |
| 缺少夜间使用可能 | 廊架 + 树下布置暖光 LED（太阳能），延长公园使用时间至傍晚 |

### 3.3 人流分析

```
      村路（行人/电动自行车）
   ─────────────────────────────
        ↑     ↑     ↑
        │     │     │
   ┌────┼─────┼─────┼────┐
   │    │     │     │    │
   │  入口  棋桌  入口  │
   │        廊架        │
   │    童趣角  花池   │
   │                    │
   └────────┬───────────┘
            │
      党群服务中心西门
      （主要人流来源）
```

三股人流在场地交汇：
1. **办事人流**：从西门出 → 廊架等候/看公告 → 离开
2. **路过人流**：沿村路 → 被棋桌/廊架吸引 → 进入
3. **专程人流**：本村居民 → 带小孩 → 童趣角 + 树下坐憩

## 四、空间布局：三环五境

### 4.1 整体结构：三环 × 五境的渐变网格

**面状布局（不同于五福村的线状布局）——以党群服务中心西门为"锚点"，向外辐射**

```
         村  路
   ┌──────────────────┐
   │  ▓▓ 境一·迎 ▓▓  │  ← 沿路入口标识 + 信息栏
   │  ▓             ▓ │
   │  境二·弈  境三·聚 │  ← 棋桌区 + 廊架核心区
   │  ▓             ▓ │
   │  ▓▓ 境四·趣 ▓▓  │  ← 童趣角 + 植物认知
   │  ▓▓ 境五·静 ▓▓  │  ← 安静坐憩 + 草坡
   └──────────────────┘
    ↑                 ↑
   西门           场地边界
```

| 节点 | 位置 | 主题 | 功能 | 强度 |
|---|---|---|---|---|
| **境一·迎** | 沿村路界面 | 入口标识 + 党群信息栏 + 共享种植 | 迎候、信息获取、认养 | ★★ |
| **境二·弈** | 保留大树下 | 棋盘桌 + 石凳 + 散座 | 下棋、围观、短暂歇脚 | ★★★ |
| **境三·聚** | 西门正对核心区 | 邻里廊架 + 候事长凳 + 公告栏 + 遮阳伞 | 等候叫号、聊天、小型议事、喝茶 | ★★★★★ |
| **境四·趣** | 场地内侧/日照较好处 | 童趣角 + 本土植物认知花池 + 自然游乐 | 亲子、认知、玩耍 | ★★★ |
| **境五·静** | 场地最内侧 | 草坡坐憩 + 独坐长凳 + 驱蚊植物丛 | 静坐、发呆、独处 | ★★ |

> 活动强度从西门向外逐渐变化：`西门（最"正式"）→ 廊架（过渡）→ 草坪/树下（最"放松"）`

### 4.2 各节点详细设计

#### 境一「迎」—— 沿路入口标识 + 党群信息栏

| 元素 | 描述 | 尺寸/位置 |
|---|---|---|
| **入口标识矮墙** | 回收旧青砖错缝砌筑，顶面旧木压顶，正面嵌铜字「三角·邻里汇」，夜间底部埋地灯洗亮 | 2.0m(L) × 0.3m(W) × 0.7m(H)，位于沿路界面中部 |
| **党群信息栏** | 青砖框架 + 可翻转木百叶面板（双面使用：A 面党群公告/B 面村务公开），顶部水蓝色遮雨檐，内置太阳能 LED 灯带。翻板设计——村民走过顺手翻一翻，公告"活"起来 | 1.2m(L) × 0.15m(W) × 1.6m(H)，位于标识墙旁 |
| **共享种植角** | 3 个抬升种植箱沿场地边界排列，回收旧门板拼制。种植斗门本土香草（紫苏/香茅/红葱头/灯盏菜），侧板插可擦写亚克力名牌（"阿芳的紫苏""明叔的灯盏菜"），与党群服务中心"服务"主题呼应——种的是菜，也是邻里情 | 每箱 0.6m × 1.0m × H0.7m，总长约 3.5m |
| **地面** | 浅灰白透水砖（环一·迎候环起点） | — |

#### 境二「弈」—— 大树下棋盘桌 + 散座

| 元素 | 描述 | 尺寸/位置 |
|---|---|---|
| **榕荫棋盘桌** | 斗门本地花岗岩整石切割，桌面激光雕刻象棋棋盘格 + 三角几何纹样边框，四角各一个杯槽（不锈钢内胆可取）。桌腿由 4 墩旧青砖砌成 | 0.8m × 0.8m × H0.7m，位于保留大树南侧，距树干 1.5m |
| **石凳 ×4** | 本地花岗岩切块，顶面荔枝面防滑处理 | 0.35m × 0.35m × H0.45m |
| **树下散座** | 围绕树干的 3 段弧形旧木拼制长凳（无靠背，双面可坐），120° 分布 | 每段 1.2m(L) × 0.4m(W) × H0.45m |
| **地面渐变** | 环一（透水砖）与环二（碎石+草缝）在此处出现第一条"模糊交界线" | — |

#### 境三「聚」—— 邻里廊架 + 候事长凳 + 核心广场

这是全场的 **核心区**，正对党群服务中心西门。来办事的人在廊架下翻报纸等叫号，邻居在廊架另一侧喝茶聊天——同一个廊架，两面不同的生活。

| 元素 | 描述 | 尺寸/位置 |
|---|---|---|
| **「邻里廊架」** | 双面使用 L 形廊架。骨架：热镀锌方钢（深灰氟碳喷涂），顶棚：回收旧木格栅 + 局部水蓝色 Sunbrella 帆布（可拆卸，台风季收入储物箱）。廊架内设双面长凳——面朝西门侧为"候事区"（可坐 6-8 人等叫号），面朝公园内侧为"聊天区"（可坐 6-8 人面向草坪）。廊架侧面设可翻转公告栏（与境一信息栏同款）。廊架四角立柱底部嵌暖光 LED（3000K） | 4.5m(L) × 2.4m(W) × H2.4m，位于西门正前方约 8-10m |
| **蚝壳水磨石圆形小广场** | 廊架前方，直径 3.0m 圆形铺装区——全场唯一使用蚝壳碎的区域（与水磨石工艺结合，表面打磨+荔枝纹防滑）。圆心嵌入三角村 Logo 铜标（三角形轮廓 + 蚝壳纹样 + "三角"字） | 直径 3.0m，位于廊架南侧 |
| **遮阳伞 ×2** | 水蓝色→莲雾粉渐变伞面，伞缘垂下短布帘（丝印咸水歌五线谱）。一把在蚝壳广场旁，一把在廊架旁。伞顶太阳能板供电，夜间伞柱暖光 LED 自动亮起 | 伞径 2.5m，伞高 2.4m |
| **候事长凳** | 廊架外侧靠西门方向，2 张独立旧木长凳，面向西门——"一眼就看到叫到几号了" | 每张 1.5m(L) × 0.45m(W) × H0.45m |

#### 境四「趣」—— 童趣角 + 本土植物认知花池

| 元素 | 描述 | 尺寸/位置 |
|---|---|---|
| **「斗门植物小课堂」花池** | 3 个不同高度的圆形抬升花池（青砖砌筑 + 旧木压顶），分别种植斗门本土特色植物：① 可触摸香草类（香茅/薄荷/紫苏）；② 可食用类（红葱头/灯盏菜）；③ 可观赏类（莲雾小树/万寿菊）。每个花池前插一块小木牌，手写植物名称 + 粤语俗称 + 用途 | 直径 0.8m / 1.0m / 1.2m，高度 0.3m / 0.5m / 0.7m（适合不同年龄段儿童），总占地约 12㎡ |
| **自然游乐元素** | 旧木桩平衡木（3 根荔枝木粗枝横卧地面，不同直径） + 碎石子游戏区（可赤脚踩的细碎石片，约 4㎡） | 散布于花池之间 |
| **亲子长凳** | 一张旧木长凳，面向花池和游戏区，家长坐着就能看到孩子 | 1.8m(L) × 0.45m(W) × H0.45m |
| **地面** | 树皮/木屑 + 细碎石混合，柔软防摔 | — |

#### 境五「静」—— 草坡坐憩 + 独处角

| 元素 | 描述 | 尺寸/位置 |
|---|---|---|
| **微地形草坡** | 利用场地内侧角落堆出约 0.5m 高的缓坡（坡度 1:4），铺本地草种（结缕草），坡顶嵌 2 块扁平旧石板——可坐可躺。"有坡的公园才是好公园" | 约 15-20㎡，位于场地东南角 |
| **独坐长凳** | 一张回收旧木长凳，背靠花池面向草坡，适合一个人安静坐憩 | 1.8m(L) × 0.45m(W) × H0.45m |
| **驱蚊植物丛** | 沿场地边界丛植香茅 + 迷迭香 + 艾草 + 万寿菊 | 约 5㎡ 自然式丛植 |
| **咸水歌木篱** | 3 根回收荔枝木粗枝立柱，顶部悬挂镂空锈板——蚀刻咸水歌《海底珍珠容易揾》选段。风吹锈板微晃发出低沉"笃笃"声 | 立柱 H1.5m，间距 1.2m |

### 4.3 空间渐变总表

| 维度 | 境一·迎 | 境二·弈 | 境三·聚 | 境四·趣 | 境五·静 |
|---|---|---|---|---|---|
| **铺装** | 100%透水砖 | 70%砖+30%碎石 | 50%碎石+30%砖+**20%蚝壳水磨石** | 50%树皮+30%碎石+20%草坪 | 70%草坪+30%树皮 |

## 五、铺装体系

（兴安广场最具参考价值的技法——**铺装不换，空间不分**）

### 5.1 铺装渐变平面

```
村路（沥青路面）
═══════════════════════════════════════════════════════
环一·迎候环 · 透水混凝土砖（浅灰白 #F2EFE9 仿蚝壳色）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

环二·社交环 · 碎石+草缝，局部蚝壳水磨石圆形广场
     ← 碎石比例降低，草缝变宽，树皮开始出现 →
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
环三·沉浸环 · 草坪 + 树皮/木屑覆盖
═══════════════════════════════════════════════════════
场地内边界
```

### 5.2 铺装类型与面积

| 铺装类型 | 应用区域 | 面积(估) | 占比 |
|---|---|---|---|
| 透水混凝土砖（浅灰白仿蚝壳色） | 环一全线 + 沿路入口区 | 约 90㎡ | 30% |
| 白麻碎石 + 草缝 | 环二主体（境二 ~ 境四） | 约 110㎡ | 37% |
| **蚝壳碎+树脂水磨石（精装）** | 仅境三中央圆形广场（直径 3.0m） | 约 7㎡ | 2% |
| 树皮/木屑覆盖 | 境四童趣区 + 树根保护区 | 约 35㎡ | 12% |
| 草坪（结缕草） | 境五草坡 + 环三部分 | 约 58㎡ | 19% |
| **合计** | | **约 300㎡** | **100%** |

> 蚝壳实际用量：仅 7㎡ × 3cm 厚 ≈ 0.21m³，约需蚝壳碎 150-200kg。一个海鲜餐厅约 5-7 天的蚝壳量。

### 5.3 无路牙设计（关键细节）

- 村路边与场地之间 **不设路牙石**
- 透水砖从路边以 2% 坡度向公园内微倾，雨水自然排入碎石带下渗
- 三种铺装之间不设金属收边条——碎石自然"渗入"砖缝，草坪自然"蔓延"进碎石区
- 党群服务中心西门出口地面与公园铺装同色衔接——"出门就是公园"

## 九、落地性专题与后期运营管理

### 9.1 蚝壳铺装可行性（定论）

| 问题 | 风险 | 本方案对策 |
|---|---|---|
| 蚝壳碎片尖锐易划伤 | 大面积铺装摔伤风险高 | **仅用于 7㎡ 圆形广场水磨石**，碎片被树脂完全包裹，表面打磨+荔枝纹防滑处理 |
| 雨后打滑 | 行人滑倒 | 水磨石表面荔枝面处理 + 位于廊架/遮阳伞下，雨天较少淋雨 |
| 碳酸钙粉化 | 耐久性差 | 树脂包裹隔绝水分和酸性雨水，耐久性等同于水磨石地面（10 年+） |
| 清洁困难 | 壳缝藏污纳垢 | 仅 7㎡，日常人工清扫即可 |
| **大面积替代方案** | — | 其他 290+㎡ 使用同色系透水砖 + 本地碎石 + 草坪，以色彩而非材质延续"蚝壳白"基因 |

## 十一、Remotion 动画脚本（React + TypeScript）

### 11.1 使用说明

- 将 3D 模型文件（.glb）放入 `public/` 目录
- 修改 `MODEL_PATH` 常量为模型文件名
- 运行 `npx remotion render` 输出 15 秒 MP4

### 11.2 项目初始化

```bash
npx create-video@latest sanjiao-animation
cd sanjiao-animation
npm install @remotion/three @react-three/fiber @react-three/drei three
```

### 11.3 `src/Root.tsx`

```tsx
import { Composition } from "remotion";
import { SanjiaoScene } from "./SanjiaoScene";

export const RemotionRoot: React.FC = () => {
  return (
    <Composition
      id="SanjiaoNeighborhoodHub"
      component={SanjiaoScene}
      durationInFrames={450}
      fps={30}
      width={1920}
      height={1080}
      defaultProps={{
        modelPath: "public/sanjiao-park.glb", // ← 替换为你的模型路径
      }}
    />
  );
};
```

### 11.4 `src/SanjiaoScene.tsx`

```tsx
import React, { useRef, Suspense } from "react";
import {
  useCurrentFrame,
  useVideoConfig,
  AbsoluteFill,
  interpolate,
} from "remotion";
import { Canvas, useFrame, useLoader } from "@react-three/fiber";
import { Environment, ContactShadows } from "@react-three/drei";
import { GLTFLoader } from "three/examples/jsm/loaders/GLTFLoader";
import type { Group } from "three";

const Model: React.FC<{ modelPath: string; rotationSpeed: number }> = ({
  modelPath,
  rotationSpeed,
}) => {
  const groupRef = useRef<Group>(null);
  const gltf = useLoader(GLTFLoader, modelPath);

  useFrame(() => {
    if (groupRef.current) {
      groupRef.current.rotation.y += rotationSpeed;
    }
  });

  return (
    <group ref={groupRef}>
      <primitive object={gltf.scene} />
    </group>
  );
};

const ThreeScene: React.FC<{ modelPath: string; rotationSpeed: number }> = ({
  modelPath,
  rotationSpeed,
}) => {
  return (
    <Canvas
      camera={{ position: [8, 7, 14], fov: 40 }}
      shadows
      style={{ width: "100%", height: "100%" }}
    >
      <ambientLight intensity={0.6} />
      <directionalLight
        position={[10, 15, 5]}
        intensity={1.2}
        castShadow
        shadow-mapSize-width={2048}
        shadow-mapSize-height={2048}
      />
      <hemisphereLight args={["#FFF7E6", "#5B8C7A", 0.4]} />
      <Suspense fallback={null}>
        <Model modelPath={modelPath} rotationSpeed={rotationSpeed} />
      </Suspense>
      <ContactShadows
        position={[0, -2.5, 0]}
        opacity={0.4}
        scale={20}
        blur={2}
        far={4}
      />
      <Environment preset="sunset" />
    </Canvas>
  );
};

interface SanjiaoSceneProps {
  modelPath: string;
}

export const SanjiaoScene: React.FC<SanjiaoSceneProps> = ({ modelPath }) => {
  const frame = useCurrentFrame();
  const { fps } = useVideoConfig();
  const totalFrames = fps * 15;
  const fadeInEnd = fps * 1;
  const fadeOutStart = fps * 13.5;

  const sceneOpacity = interpolate(
    frame,
    [0, fadeInEnd, fadeOutStart, totalFrames - fps * 0.5, totalFrames],
    [0, 1, 1, 0, 0]
  );

  const rotationSpeed = 0.004;

  const titleOpacity = interpolate(
    frame,
    [fadeInEnd, fps * 2, fadeOutStart, fadeOutStart + fps * 0.5],
    [0, 1, 1, 0]
  );
  const titleY = interpolate(
    frame,
    [fadeInEnd, fps * 2],
    [20, 0]
  );

  return (
    <AbsoluteFill style={{ backgroundColor: "#F2EFE9" }}>
      <div style={{ width: "100%", height: "100%", opacity: sceneOpacity }}>
        <ThreeScene modelPath={modelPath} rotationSpeed={rotationSpeed} />
      </div>

      <div
        style={{
          position: "absolute",
          bottom: "8%",
          left: 0,
          right: 0,
          display: "flex",
          flexDirection: "column",
          alignItems: "center",
          gap: 10,
          opacity: titleOpacity,
          transform: `translateY(${titleY}px)`,
          pointerEvents: "none",
        }}
      >
        <h1
          style={{
            fontFamily: '"思源宋体 CN", "Noto Serif SC", serif',
            fontSize: 54,
            fontWeight: 700,
            color: "#3C6E71",
            margin: 0,
            letterSpacing: "0.12em",
            textShadow: "0 2px 20px rgba(242, 239, 233, 0.85)",
          }}
        >
          三角村 · 邻里汇
        </h1>
        <p
          style={{
            fontFamily: '"思源黑体 CN", "Noto Sans SC", sans-serif',
            fontSize: 22,
            fontWeight: 400,
            color: "#8B6914",
            margin: 0,
            letterSpacing: "0.06em",
          }}
        >
          口袋公园设计 —— 珠海斗门三角村党群服务中心西侧
        </p>
      </div>
    </AbsoluteFill>
  );
};
```

> 本方案以"三角·邻里汇"为主题，将党群服务中心的户外空间转化为兼具办事等候、邻里社交、儿童友好和文化感知的口袋公园。以"三环渐变"（迎候环 → 社交环 → 沉浸环）替代五福村的线性渐变，以"双面廊架"回应党群服务中心的特殊功能需求，以"三角几何母题"与村名呼应，是一份具有高落地性、强运营逻辑和在地文化厚度的微景观设计方案。
