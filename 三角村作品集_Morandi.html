项目概况 Overview

六棵树，六段渐渐慢下来的时光。窄不是问题，关键是让空间自己"说话"——铺装从硬质透水砖慢慢变成碎石草缝，再变成树皮木屑；家具从路边的一张石凳，慢慢变成树下的棋盘桌，再变成围着树干的一圈卡座。

本项目位于珠海市斗门区三角村沿路带状绿地，场地面积约 300㎡（长约 26m，进深约 7m），保留 6 棵沿路乔木。设计核心借鉴湖北宜昌兴安广场（HID 翰地景观）的「渐变空间」理念，将其从城市商业语境转译为岭南乡村公共空间——以铺装密度、家具高度、植物郁闭度的三向渐变，取代传统硬质围墙分隔。

设计概念 Design Concept

兴安广场 → 三角村渐变转译

渐变维度兴安广场（城市版）三角村·三环五境（乡村版）
铺装渐变 市政人行道透水砖 → 渐变拼色铺装 → 木平台 → 室内地面 透水砖 → 碎石草缝 → 蚝壳水磨石圆广场 → 树皮木屑
家具密度渐变 沿路散点座椅 → 中段组合卡座+吧台 → 内区围合式卡座 路边石凳 → 树下棋盘桌 → 茶寮推车+遮阳伞 → 环抱卡座
活动强度渐变 外圈通过/驻足 → 中圈社交/消费 → 内圈沉浸/停留 入口平缓 → 中段社交高峰 → 东端静坐回落（像一个深呼吸）
色调体系 红灰工业色调 蚝壳白 + 青砖灰 + 老木褐 + 水色蓝绿 + 疍家土黄 + 莲雾粉
核心家具 咖啡馆固定吧台 「三角茶寮」可折叠推车（村民轮值）
主要客群 城市青年 老年下棋 / 青年打卡 / 儿童认知 / 路人歇脚，全龄共用

六棵树，六段渐渐慢下来的时光。从西到东，铺装从硬质透水砖慢慢变成碎石草缝，再变成树皮木屑；家具从路边的一张石凳，慢慢变成树下的棋盘桌，再变成围着树干的一圈卡座，最后是一把遮阳伞下的茶寮推车——一个不想走的人，自然会沿着这六棵树走进去、坐下来、聊起来。

蚝壳白的铺地、青砖灰的矮墙、水蓝色的伞面、莲雾粉的招牌……斗门本土的六种颜色不吵不闹地散落在这条"渐境"里。阿伯在第二棵树下下棋，阿婆在第四棵树下剥花生，年轻人在第五棵树前举着莲雾汁拍照——六棵树，把一段普通的村路边角地，变成了全村最有温度的一排"檐下座"。

6 棵树 = 6 个空间锚点

每棵树自带遮荫半径（约 3–5m）、"树下领域感"和视觉焦点属性。设计原则：不移动任何一棵树，以每棵树为中心组织一个"微空间"，6 个微空间之间不设硬隔断，而是用铺装纹理和家具密度形成渐变。

现状问题 → 设计对策

现状问题渐变策略
长条形 → 行人径直路过，不停留用铺装纹理的"方向性渐变"（横向条纹→散点→无方向）暗示减速
仅 6–8m 进深，做不了大广场不做"面状聚集"，做"线状串联"——沿树列形成 6 个口袋空间
6 棵树看似均匀，实际间距不等中段 2 棵树间距最大处放大为"聚场"，两端收拢为"静角"
路边界面生硬（路牙+裸土）取消路牙，铺装从路边"渗入"树根区——让道路和绿地之间没有那条线

三环五境 · 空间布局 Six Trees · Six Realms

三带 × 六境 的渐变网格

纵向（垂直于村路）：3 条渐变带——Layer A 通行带（透水砖，全公共）→ Layer B 坐憩带（碎石+草缝，半公共）→ Layer C 沉浸带（树皮/木屑，半私密）。三条带之间没有任何硬质边界，铺装的"纹理密度"和家具的"坐面高度"是唯一的分界语言。

横向（沿村路）：6 棵树 = 5 个渐境节点，活动强度从西到东呈"深呼吸"曲线：入口平缓 → 中段高峰 → 东端回落。

铺装渐变体系 Paving Gradient

这是兴安广场最具参考价值的技法——铺装不换，空间不分。三种铺装之间不设金属收边条，碎石自然"渗入"砖缝，树皮自然"蔓延"进碎石区。路边与场地之间不设路牙石——你还没决定"要不要进去"，就已经进去了。

铺装类型应用区域面积（估）占比
透水混凝土砖（浅灰白仿蚝壳色）Layer A 全线 + 境一/境二 Layer B 部分约 55㎡35%
白麻碎石 + 草缝Layer B 主体（境二 ~ 境五）约 65㎡40%
蚝壳碎 + 树脂水磨石（精装）仅境三中央圆形广场（直径 2.4m）约 4.5㎡3%
树皮/木屑覆盖Layer C 全线 + 所有树根保护区约 35㎡22%

蚝壳实际用量：仅 4.5㎡ × 3cm 厚 ≈ 0.14m³，约需蚝壳碎 100–150kg。一个海鲜餐厅约 3–5 天的蚝壳量即可满足。

运营管理 Operations & Maintenance

运营模式

「村委会监管 + 村民家庭轮值」微经营模式。本村有意愿的村民家庭以"周"为单位轮值，优先长者/低收入家庭。仅售 3–4 种饮品（不需明火烹饪），统一定价 5 元/杯。轮值家庭得 80%，20% 上交村委会作为场地维护基金。非营业时段推车折叠锁闭转为景观装置 + 夜间暖灯。

小场地独特优势

  • — 造价低：约 3–5 万即可完成，适合村集体自筹+小额奖补
  • — 施工快：约 2–3 周可完工，不扰民
  • — 维护轻：面积小 = 清扫量小、设备少 = 故障点少
  • — 可复制：「六棵树模型」可在斗门任意沿路带状绿地复制推广
  • — 亲和力强：小空间不空旷、三五个人就"满"了，更容易形成"有人气"的良性循环
