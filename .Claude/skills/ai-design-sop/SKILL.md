# AI 辅助设计 SOP — 全流程自动化工作手册

## 触发条件

用户输入涉及以下任一内容时激活：
- 建筑设计/景观设计/规划设计 + AI辅助
- AutoCAD/SketchUp/Photoshop + 自动化/脚本/批处理
- Computer Use 控制桌面软件
- MCP 服务器控制 SketchUp
- .scr/.jsx/.rb 脚本生成
- 参数化建模/批量出图

## 核心流程

### 阶段一：CAD 底图

**交互式控制**：Codex Computer Use 直接操作 CAD
```
在AutoCAD中：
1. 新建文件，设置单位为米
2. 创建图层：道路(红色)、建筑(蓝色)、绿化(绿色)
3. 绘制[尺寸]的矩形作为用地红线
4. 保存为[文件名].dwg
```

**脚本自动化**：生成 .scr 脚本或 Python pyautocad 脚本

### 阶段二：SketchUp 建模

**MCP 工作流**：Claude → su_mcp.rb → SketchUp
```
通过MCP执行：
1. 导入'[底图].dwg'
2. 将图层'建筑'的闭合多段线推拉[高度]米
3. 为每个体块创建组件
4. 导出为'[模型].skp'
```

**Ruby 脚本**：生成参数化构件（可复用）

### 阶段三：Photoshop 后期

**Computer Use**：直接控制 PS 调色/合成
```
在Photoshop中：
1. 打开'[渲染图].jpg'
2. Camera Raw: 曝光+0.3 对比度+15
3. 新建柔光图层添加暖色渐变
4. 保存为PSD和JPG
```

**.jsx 批处理**：批量调色/缩放/水印

## 指令工程原则

1. 环境状态 → 2. 动作分解（数字序号） → 3. 参数具体（精确数值） → 4. 输出指定

| 模糊 | 精确 |
|-----|------|
| "做个好看的设计" | "饱和度+15%，对比度+10%" |
| "优化一下" | "锐化80%，半径1.2px" |
| "填绿色" | "填充#8BC34A，叠加纹理50%" |

## 质量控制

每次 AI 操作后检查：尺寸精度/图层组织/文件链接/输出格式

## 风险边界

- AI 能做：重复绘图、参数化建模、批量处理、格式转换
- 人工必须：创意概念、规范审查、结构判断、审美决策、最终签字
